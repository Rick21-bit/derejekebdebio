# Dereje Kebede Bio

A Vue landing page for Dr. Dereje Kebede, an Ethiopian gospel singer whose music became my favorite in this season.

## Live site

Hosted on GitHub Pages.

## Data

Biography and discography are based on public sources, with links to the WikiMezmur lyrics database for full song lyrics.

## Notes

- Built with Vue 3 via CDN so it runs from a static host.
- Lyrics are not embedded to respect rights; only song titles and external links are listed.
